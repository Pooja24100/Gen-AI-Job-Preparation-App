# Contributing

Thanks for your interest in improving GenAI Job Preparation App.

## Local Development

Backend:

```bash
cd Backend
npm install
npm run dev
```

Frontend:

```bash
cd Frontend
npm install
npm run dev
```

## Pull Request Checklist

- [ ] Backend changes are manually tested
- [ ] Frontend changes are manually tested
- [ ] Environment variables are documented
- [ ] No secrets are committed
- [ ] README/API documentation is updated if needed

## Commit Style

Use Conventional Commits:

- `feat: add interview report export`
- `fix: secure auth cookie settings`
- `docs: improve API documentation`
- `refactor: centralize error handling`
